🔐 เครื่องมือเข้ารหัสกุญแจ Sync

สำหรับเปลี่ยน token ที่ฝังในเว็บแอป (Audit CRM / Audit Tracker) — ทุกอย่างทำงานในเครื่องนี้เท่านั้น ไม่มีการส่ง token หรือ PIN ออกไปที่ไหนทั้งสิ้น (ปิดเน็ตแล้วใช้ก็ยังได้ ยกเว้นปุ่มทดสอบ token)
🚫 ห้ามส่ง token (github_pat_/ghp_) หรือ PIN ให้ใครทั้งนั้น รวมถึงใน chat — สิ่งเดียวที่ส่งต่อได้คือ "ผลลัพธ์ที่เข้ารหัสแล้ว" จากกล่องล่างสุด
ขั้นตอนสร้าง token แบบไม่หมดอายุ
  1. เปิด github.com/settings/personal-access-tokens/new (ล็อกอิน tpnaudit)
  2. Token name: audit-sync-forever · Expiration: No expiration (ถ้าไม่มีตัวเลือกนี้ ให้ใช้ classic token ที่ github.com/settings/tokens/new เลือก No expiration + scope repo)
  3. Repository access → Only select repositories → wst-audit-data
  4. Permissions → Contents → Read and write
  5. Generate แล้ว copy มาวางด้านล่าง
✅ ผลลัพธ์ (เข้ารหัสแล้ว — ก้อนนี้ส่งต่อได้ปลอดภัย)
ระบบทดสอบถอดรหัสกลับด้วย PIN เดิมแล้วตรงกัน — copy ทั้งกล่องไปส่งให้ผู้ดูแลอัพเดทไฟล์ sync-token.enc.js ได้เลย
